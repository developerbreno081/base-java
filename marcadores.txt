Tipos de marcadores:

%d - valores inteiros
 
%f - valores decimais

%s - valores texto

Exemplo:

// declarando variáveis
int a  = 21; | double b = 100.0; | String c = "Breno";

// exibindo valores com os seus respectivos marcadores
System.out.printf("Meu nome é %s e tenho %d. Meu salário atual é %.2f ", c, a, b);

// %.2f = aqui é um bônus indicando como pode-se exibir o número com 2 casas decimais por exemplo
